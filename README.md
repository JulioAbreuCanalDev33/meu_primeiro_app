#📦 Como Rodar o Projeto

Pré-requisitos

1. Instale o Flutter SDK (versão 3.0 ou superior).


2. Configure seu ambiente de desenvolvimento conforme a documentação oficial do Flutter.



Passos


1. Clone este repositório:


git clone https://github.com/JulioAbreuCanalDev33/meu_primeiro_app.git  
cd meu_primeiro_app


2. Instale as dependências:


flutter pub get


3. Execute o projeto em um emulador ou dispositivo conectado:


flutter run




---

🎯 Como Usar

1. Preencha os campos:


Valor inicial de aporte (R$)

Valor mensal de aporte (R$)

Taxa de juros anual (%)

Tempo investido (meses)



2. Calcule:

Clique no botão Calcular para visualizar:


Capital Investido

Juros Acumulados

Montante Final




3. Modo Claro/Escuro:


Alterne entre os modos usando o botão no canto inferior direito.
